# Agent Instructions

## Package Manager
- Frontend: **npm** `npm install`, `npm run dev`, `npm run build`, `npm run lint`
- Backend: **pip** `python -m venv venv`, `pip install -r requirements.txt`

## Commit Attribution
- AI commits MUST include:
```
Co-Authored-By: Codex <noreply@example.com>
```

## Key Conventions
- Frontend app lives in `frontend/`
- Backend app lives in `backend/`
- Backend env file: `backend/.env` (copy from `backend/.env.example`)
